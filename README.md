# Simvy — מסמכים משפטיים (אתר סטטי)

אתר סטטי קל לאחסון המסמכים המשפטיים של אפליקציית Simvy. מתארח ב-GitHub Pages.

## דפים
- `index.html` — דף נחיתה עם קישורים לשלושת המסמכים.
- `privacy.html` — מדיניות פרטיות.
- `terms.html` — תקנון ותנאי שימוש (כולל ההבהרה המשפטית — סעיף א').
- `delete-account.html` — דף מחיקת חשבון (דרישת חנות; נגיש ללא התחברות).
- `styles.css` — עיצוב משותף (RTL עברית, ללא תלות בספריות חיצוניות).

## כתובות (URLs) לאחר פרסום ב-GitHub Pages
מבנה: `https://<USER>.github.io/simvy-legal/<page>.html`

| מסמך         | קובץ                  | משתנה סביבה באפליקציה              |
|--------------|-----------------------|-----------------------------------|
| פרטיות       | `privacy.html`        | `EXPO_PUBLIC_PRIVACY_URL`          |
| תקנון        | `terms.html`          | `EXPO_PUBLIC_TERMS_URL`            |
| מחיקת חשבון  | `delete-account.html` | `EXPO_PUBLIC_ACCOUNT_DELETION_URL` |

## עדכון תוכן (אחרי סקירת עו"ד)
כל התוכן נמצא ישירות בקבצי ה-HTML. לעריכה:
1. ערוך את הטקסט בקובץ הרלוונטי.
2. עדכן את "עדכון אחרון" בראש הדף ואת התאריך ב-`index.html`.
3. `git add -A && git commit -m "..." && git push` — GitHub Pages מתעדכן אוטומטית.
